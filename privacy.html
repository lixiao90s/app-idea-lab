隐私政策

最后更新：2026年6月11日

我们收集的信息

App Idea Lab 为纯前端分析工具。我们不注册账户，不主动收集个人身份信息。搜索关键词仅在您的浏览器本地缓存，用于加速重复查询。

第三方服务

  • Apple iTunes API：获取公开 App 数据
  • Cloudflare：托管网站与 API 代理缓存
  • Google AdSense（如启用）：可能使用 Cookie 展示个性化广告
  • MyMemory 翻译 API：仅在您点击翻译时发送评论文本

本地存储

我们使用浏览器 localStorage 缓存 API 结果（最长 6 小时），减少重复请求。

广告与 Cookie

启用广告后，第三方广告商可能使用 Cookie。您可在浏览器设置中管理 Cookie，或通过 Google 广告设置 opt-out。

联系我们

如有隐私相关问题，请通过 GitHub Issues 联系我们。
